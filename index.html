Nadir · Azimuth · Zenith

THE DAI

An end-to-end algorithmic trading pipeline that runs on your own computer — from raw market data, to a trained model, to live execution.

01 · The Data Factory

NADIR

Turns years of raw, high-frequency market data into clean, labeled, institutional-grade datasets — feature engineering, economic-calendar awareness, and structural labeling, all built to work with whatever symbol you point it at.

View on GitHub →

02 · The Training Ground

AZIMUTH

Takes Nadir's datasets and trains real machine learning models on them — with walk-forward validation, forensic auditing, and backtesting tools to prove a model deserves to go live before it ever touches real money.

View on GitHub →

03 · The Execution Engine

ZENITH

Connects a trained model to a live MetaTrader 5 account — building the same features live, applying risk controls, and routing trade decisions back to the terminal automatically.

View on GitHub →

THE DAI runs locally on your own machine and requires your own MetaTrader 5 account with a broker. Nothing here is hosted or run on our servers.

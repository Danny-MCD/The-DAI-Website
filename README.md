# The-DAI-Website

The public landing page for **THE DAI** — a single static page (no build step,
no framework) linking out to the three component repos:

- [The-DAI-NADIR](https://github.com/Danny-MCD/The-DAI-NADIR) — the data factory
- [The-DAI-AZIMUTH](https://github.com/Danny-MCD/The-DAI-AZIMUTH) — the training ground
- [The-DAI-ZENITH](https://github.com/Danny-MCD/The-DAI-ZENITH) — the execution engine

## Hosting

Meant to be served for free via GitHub Pages (Settings → Pages → deploy from
`main` branch, root folder). Once enabled it's live at:

```
https://danny-mcd.github.io/The-DAI-Website/
```

## Local preview

Just open `index.html` directly in a browser — it's plain HTML/CSS, nothing
to install or build.
